# Chess

## Run chess
Go into the top directory and run:
<hr />
<br>

On Windows:
```
buildrun.bat
```
<hr />
<br>

On Mac:
```
sh buildrun.sh
```
<hr />
<br>

On Linux:
```
./buildrun.sh
```  
<br>
<br>

## Dependencies
On Windows:  
Install CMake and add it to PATH
<hr />
<br>

On Mac:  
Install CMake and SDL2 by running
```
brew install sdl2
brew install cmake
```
<hr />
<br>

On Linux - Ubuntu:  
Install CMake and SDL2 by running
```
sudo apt-get install libsdl2-dev
sudo apt-get install cmake
```
